# Birthday H5 Assets

画布尺寸：946 × 2048 px。

重要说明：
这些图片来自整张设计图的近似切图，不是 PSD/Figma 分层源文件。
因此，透明 PNG 中的元素区域会带有少量原背景，背景图也未真正抹除可交互元素。
建议：
1. 前端先用这些 full-canvas transparent PNG 进行 1:1 绝对定位还原。
2. HTML 文案尽量用 DOM/CSS 重排，不要依赖图中文字。
3. 最终上线前，最好把小狗、印章、灯、饭菜、签牌等元素重新单独生成干净透明 PNG。
4. layout.json 记录每个元素在 946 × 2048 画布里的近似坐标。
